セットアップ編

Raspbian Stretch時点でのフロー。更新の際に一部変更になることがある。 過去にあった変更としては、hostnameの設定方法やIP固定の手順などが変更になっている。

  1. https://www.raspberrypi.org/downloads/からimageをDL
  2. 念のためSD Memory Card Formatterで消去してWin32 Disk Imagerで書き込み
  3. /boot/ssh というファイルを作成する。
    これがあるとデフォルトでSSHが有効になりディスプレイなしセットアップが可能。
  4. LANケーブルを接続して電源を接続。
  5. ルーターの管理画面からMACアドレスがB8:27:EBで始まる新規のデバイスを探す。
  6. とりあえずルータ側から固定IPを振る。このwikiにメモするとよい。
  7. sudo raspi-config から一通り設定
  8. sudo apt-get update と sudo apt-get upgrade
  9. sudo apt-get install vim
  10. IPを固定してhostnameを設定(OSのバージョンによって操作が違う)
  11. sudoなユーザを追加しpiを削除
  12. piはnopasswdなので必要に応じてvisudoで後処理
  13. 鍵交換

オプション編
